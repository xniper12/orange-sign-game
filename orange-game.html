GitHub Hosting Guide for Your Orange Sign Game

Follow these steps to host your game on GitHub Pages for FREE and get a shareable link.

Step-by-Step Instructions

1Create a GitHub Account

If you don't have one, go to github.com/signup and create a free account.

2Create a New Repository

After logging in, click the "+" icon in the top right and select "New repository".

Name it "orange-sign-game" (or any name you prefer).

Make sure to set it to "Public" and check "Add a README file".

3Upload Your HTML File

In your new repository, click the "Add file" button and select "Upload files".

Drag and drop your HTML file or click to choose it from your computer.

Name the file index.html (this is important for GitHub Pages).

4Enable GitHub Pages

Go to your repository's "Settings" tab.

In the left sidebar, click "Pages".

Under "Source", select "Deploy from a branch".

Under "Branch", select "main" and folder "/ (root)".

Click "Save".

5Get Your Game Link

Wait a few minutes (it might take up to 5 minutes to deploy).

Refresh the GitHub Pages settings page.

You'll see a message: "Your site is published at..." with your link!

Your link will look like: https://yourusername.github.io/orange-sign-game/

Congratulations! Your game is now live on the internet. Share the link with anyone to play your game!
